# Analytical Framework

Every funnel analysis in this repo follows a consistent PM lens:

**1. User Jobs-to-be-Done** — What is the advertiser trying to achieve? What is the end user experiencing?

**2. Funnel Leak Identification** — Where does the product lose users? What's the measurable drop-off?

**3. Ad Product Fit** — Which ad format/channel solves the leak? Why that one and not others?

**4. Attribution & Measurement** — How would you know if it worked? What's the north star metric?

**5. Trade-offs** — What does this approach optimize for, and what does it sacrifice?
